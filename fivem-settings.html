Load your FiveM graphics settings to get started.

File: %APPDATA%\CitizenFX\gta5_settings.xml
If the file doesn't exist, open FiveM → Settings → Graphics, change any option, then close FiveM.
Make sure FiveM is fully closed before saving
FiveM writes back to gta5_settings.xml when you exit the graphics menu or quit the game — this will overwrite your saved settings. Close FiveM completely first, then save here, then relaunch FiveM without opening Graphics Settings.
